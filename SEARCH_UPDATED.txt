WHAT'S NEW:
═══════════════════════════════════════════════════════════════

✅ Ctrl+R - Initial Search
   • Enter your search term
   • Finds first match from cursor position
   • Shows: "Found at line 15, column 5 (Ctrl+N for next)"

✅ Ctrl+N - Find Next Match ⭐ NEW!
   • Finds next occurrence of same search
   • No need to retype the query
   • Keeps cycling through all matches
   • Wraps around to beginning
   • Shows: "Found at line 23, column 12 (Ctrl+N for next)"

HOW TO USE:
═══════════════════════════════════════════════════════════════

EXAMPLE: Find all "main" in your code

Your code has:
   Line 3:  void main_func() {
   Line 4:      int main_var = 0;
   Line 7:  int main() {
   Line 8:      main_func();

Step-by-step:

1. Press Ctrl+R
   Search: main
   → Jumps to line 3, column 6
   → Message: "Found at line 3, column 6 (Ctrl+N for next)"

2. Press Ctrl+N
   → Jumps to line 4, column 9 (main_var)
   → Message: "Found at line 4, column 9 (Ctrl+N for next)"

3. Press Ctrl+N
   → Jumps to line 7, column 5 (main function)
   → Message: "Found at line 7, column 5 (Ctrl+N for next)"

4. Press Ctrl+N
   → Jumps to line 8, column 5 (main_func call)
   → Message: "Found at line 8, column 5 (Ctrl+N for next)"

5. Press Ctrl+N
   → Wraps back to line 3
   → Message: "Found at line 3, column 6 (wrapped, Ctrl+N for next)"

6. Keep pressing Ctrl+N
   → Cycles through all matches forever!
